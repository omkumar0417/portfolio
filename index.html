SRM Grade Calculator

The SRM Grade Calculator is a web-based tool developed to simplify the process of calculating grades for students. Using a clean and user-friendly interface, this application allows users to input their marks and instantly view their calculated grades, providing an efficient way to track academic performance.

This project was built using HTML, CSS, and JavaScript, showcasing my ability to combine these technologies to create an interactive and functional web application. It highlights my growing skills in front-end development and my commitment to building practical tools that address real-world needs.
